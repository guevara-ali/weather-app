# Weather Check

A small React + TypeScript app that lets you search any city and see its current temperature and wind speed, using live data from the free Open-Meteo API.

## Live Demo
[https://weather-app-ten-swart-45.vercel.app/]

## Built With
- React
- TypeScript
- Vite
- Open-Meteo API (geocoding + current weather)

## Features
- Search any city by name
- Handles loading and error states (e.g. city not found)
- Responsive, simple UI

## Running Locally
```bash
git clone https://github.com/guevara-ali/weather-app.git
cd weather-app
npm install
npm run dev
```
